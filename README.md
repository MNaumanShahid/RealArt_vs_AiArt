# RealArt vs AiArt

A Resnet152 model to classify images into RealArt or AiArt images. 

## Dataset
Dataset available on Kaggle.
### Dataset split (code in notebook):
- train_ai_img: 308
- train_real_img: 302
- test_ai_img: 77
- test_real_img: 76

## Accuracy
Accuracy of model on test data: 75.650%
